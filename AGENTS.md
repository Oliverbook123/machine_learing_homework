# CLAUDE.md

## 规则

### 用户背景
- 用户是一名研0（准研究生）学生，没有系统学习过 PyTorch。
- 用户在代码中调用的所有 PyTorch / 深度学习相关函数都需要详细注释。

### 注释规范
1. **每个 PyTorch 函数调用必须附带注释**，内容包括：
   - 该函数的**功能描述**（它本身是做什么的）。
   - 在**当前上下文中的用途**（在这里为什么要用它）。
   - 关键参数的**含义说明**（尤其是容易混淆的参数）。
2. 注释应使用**中文**书写，便于用户理解。
3. 注释形式建议使用行内注释或紧邻代码上方的块注释，保持代码可读性。

### 示例
```python
import torch

# torch.randn: 生成服从标准正态分布 N(0, 1) 的随机张量
# 这里用于创建一个形状为 (3, 4) 的随机输入矩阵，模拟一批样本数据
x = torch.randn(3, 4)  # shape: (3, 4)，3 个样本，每个样本 4 个特征

# torch.nn.Linear: 全连接/线性层，执行 y = xW^T + b 的线性变换
# 这里用于将 4 维输入特征映射到 2 维输出（二分类 logits）
layer = torch.nn.Linear(in_features=4, out_features=2)  # 4 → 2 维
```

### 交流语言
- 所有与用户的对话交流**必须使用中文**。
- 代码、变量名、函数名可以使用英文，但解释说明必须用中文。
